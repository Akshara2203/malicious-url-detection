# 🔐 Malicious URL Detection

## 📌 Overview
This project focuses on detecting malicious URLs using machine learning techniques.

## 🛠 Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn

## 🔍 Key Steps
- Data preprocessing and cleaning
- Feature extraction from URLs
- Exploratory Data Analysis (EDA)
- Model building using Random Forest

## 📊 Model Performance
- Evaluated using accuracy, precision, and recall
- Improved detection through feature engineering

## 🚀 Outcome
Developed a model capable of classifying URLs as safe or malicious, improving cybersecurity detection.
